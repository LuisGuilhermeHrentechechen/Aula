Cascating Style Sheets

Definição

Cascating Style Sheets - Folha de Estilo em Cascata - tem por objetivo realizar a estilização de conteúdo de documento HTML. Podem definir cores, fontes, espaçamentos, alinhamentos, bordas dentre outros.

Componentes

  • Seletores - selecione elementos HTML com base no nome do elemento ao qual o estilo será aplicado.
  • Propriedades - o aspecto específico do elementto que você quer usar como estilo.
  • Valores - a configuração aplicada sobre o atributo.

Formas de utilização

Inline

Os codigos CSS são definidos dentro de uma Tag HTML. Afetam somente a Tag em que foi definida.

Incorporação ou Internal

Os códigos CSS são definidos no cabeçalho, dentro da Tag head. Afetam soemnte o documento em que foi definidos.

Externo ou External

Os códigos CSS são efinidfos em um arquivo externo. Afetam todos os documentos que estão linkados ao CSS.
